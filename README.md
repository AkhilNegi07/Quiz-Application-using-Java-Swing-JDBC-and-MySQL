**📌 Project Summary**

**Quiz Application using Java Swing, JDBC, and MySQL**

Developed a GUI-based quiz application using Java Swing for an interactive user experience.
Implemented a secure login system to collect and store user information.
Integrated MySQL database using JDBC for persistent data storage and retrieval.
Designed and created relational database tables for users, quiz questions, categories, and scores.
Implemented category-based quizzes including Java, C++, Marvel, and Harry Potter themes.
Retrieved quiz questions dynamically from the database instead of hardcoding them in the application.
Developed a random question selection mechanism using SQL queries.
Implemented score calculation and result generation after quiz completion.
Stored user performance, selected category, and final scores in the database.
Applied Object-Oriented Programming (OOP) concepts such as classes, objects, methods, encapsulation, and modular design.
Utilized exception handling for robust database connectivity and runtime error management.
Designed a theme-based graphical interface with different color schemes for quiz categories.
Established a complete frontend-backend-database workflow using Java Swing, JDBC, and MySQL.
Followed modular programming practices to improve maintainability and scalability of the application.
